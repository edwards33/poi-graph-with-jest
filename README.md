## Data Structures in JS using POI and Jest (Graph Example)

  Install [POI](https://poi.js.org/):

```bash
$ npm i -g poi
```

  Install Jest:

```bash
$ npm i -D jest
```

  Install Graphdracula and Raphael:
```bash
$ npm install --save graphdracula raphael
```

  Run App and open it in a Web browser:

```bash
$ poi --serve
```

  Test Graph:

```bash
$ npm test
```
